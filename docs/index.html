核心功能

钢琴转录所需的一切

从音频到乐谱，PianoScribe 以业界领先的 AI 模型覆盖完整转录流程。

🎵

AI 音源分离

基于 Mel-Band RoFormer (pymss) 自动从混合音频中提取钢琴音轨。无论是歌曲、现场演出还是管弦乐作品，都能精准分离出纯净的钢琴声部。

🤖

深度学习转录

采用 Transkun (CRNN) 与 Basic Pitch (ONNX) 双模型架构，在标准基准测试中音符 F1 达到 0.96 以上，踏板 F1 达到 0.91 以上。

🎼

专业乐谱排版

通过 LilyPond 乐谱排版引擎生成出版级质量的乐谱。支持多声部、力度记号、演奏法标注以及标准音乐记谱法。

▶️

交互式播放

内置 FluidR3_GM 高品质音色库 (141MB) 的 MIDI 合成器。支持原始力度与统一音量两种播放模式，播放时实时高亮当前音符。

📈

难度分析

基于音符密度、手跨度、速度和技术模式自动评定难度等级（入门 / 进阶 / 高级 / 大师），是练习规划的好帮手。

🕐

节拍与速度追踪

精准的节拍检测与速度映射，支持可调 BPM。识别自由速度、渐快和渐慢，还原富有表现力的演奏风格。

工作流程

四步完成从音频到乐谱

每个环节都由前沿 AI 驱动，流程无缝衔接。

1

加载音频文件

拖放任意音频文件即可开始 — 支持 MP3、WAV、FLAC、OGG、M4A 等格式。PianoScribe 自动处理后续所有步骤，支持长达数小时的文件。

MP3 WAV FLAC OGG M4A
2

AI 音源分离

Mel-Band RoFormer 模型从混音中分离出钢琴音轨。人声、鼓点、吉他 — 全部干净分离，确保转录模型接收到纯净的钢琴音频。

Mel-Band RoFormer pymss Spleeter
3

神经网络转录

双模型流水线：Transkun CRNN 负责精确的起止时间检测，Basic Pitch ONNX 负责稳健的音高估计。两路结果融合，实现最高准确度。

Transkun CRNN Basic Pitch ONNX F1 > 0.96
4

乐谱生成与播放

MIDI 音符转换为 LilyPond 记谱法，排版为专业乐谱 (PDF)，并合成逼真的钢琴音色供即时播放。

LilyPond FluidSynth PDF 导出 MIDI 导出
